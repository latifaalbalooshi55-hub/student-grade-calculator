# Student Grade Calculator

A simple Python program that calculates a student's overall grade based on assignments, quizzes, a midterm exam, and a final exam.

## Features

- Calculates the final percentage
- Displays the letter grade
- Shows whether the student passed or failed
- Accepts grades entered by the user

## Technologies Used

- Python

## How to Run

1. Download the `grade_calculator.py` file.
2. Open it using Python or Visual Studio Code.
3. Run the program.
4. Enter the requested grades.

## Grade Weights

- Assignments: 20%
- Quizzes: 20%
- Midterm: 25%
- Final exam: 35%
